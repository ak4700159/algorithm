배낭의 크기가 작다면 답을 구하기 쉽지만 크기가 크다면 답을 구하기 어려워진다.

n개의 물건과 각 물건 i의 무게 w와 가치 v가 주어지고 배낭의 용량이 C일 때, 배낭에 담을 수 있는 물건의 최대 가치를 찾는 문제다.
